OS X新手必备

  • os x
EssaysOS X

本文为OS X系列的第二篇，也是拖了蛮久才写。

近半年来，身边多了好几台MAC ，我都不同程度的推荐了他们应该使用的系统工具/特色、快捷键以及软件等。我们就从这两方面开个头吧，后续继续分享OS X的实用经验。

首先必须得力推卖桃君的这篇文章：开始使用Mac OS X——写给Mac新人。

然后我再写写自己一些总结。

一、系统工具/特色

  1. OS X不需要每天关机，这是我从windows迁移过来的最大变化之一。在之前的文章里我也提过，扣上盖子就可以下班回家了~得益于此，那些本身启动就很慢的大型应用，比如eclipse，idea，webstorm，photoshop等，就不必反复开启关闭了，打开盖子就可以马上干活。再举一个身边的例子，很多站点的cookie有效期，默认为至浏览器关闭前，由于我们不需要经常关机，浏览器可以长期开着，那么我们再次访问这个站点时，cookie将继续有效，不需再次登录。
  2. 合理利用spotlight。使用spotlight可以搜索日常文件与应用程序，只要你记得目标的一点蛛丝马迹，都可以通过Option + Space呼出spotlight，输入关键字找到它。不过我对它的使用也不算深入，日常使用关键字搜索足够了，更多用法请自行google。
  3. 基本可以放弃使用“桌面”了，至少我觉得没有必要了。使用windows时，我们在桌面放的最多的是应用程序的快捷方式与一些临时文件,而在OS X中，应用程序可以拖到dock（类似于win7的固定到任务栏），或通过spotlight搜索，比如打开QQ的步骤：
    Option + Space ==> QQ ==> Enter，Done！完全不需要鼠标，简单高效！
    而查找文件，仍然可以通过spotlight。通常浏览器、QQ等软件的默认下载目录都是finder侧面的“下载”目录。
  4. 合理使用OS X多桌面功能。如果你一定要使用桌面的话，OS X的多桌面功能会带给你彻底分门别类的快感，点击F3，在顶部点击+号即可添加桌面，再在新桌面上添加你需要归类的软件快捷方式。通常可用于区分不同的使用场景，比如家庭桌面，工作桌面等。
  5. 充分使用OS X自带手势操作。OS X的手势操作是相当实用与灵敏的，硕大的触摸板，给我们足够的操作空间。典型的手势是：
    双指滑动(包括垂直与水平)：鼠标滚轮
    双指轻拍：鼠标右键
    三指滑动（可配置）：点住鼠标坐标并拖动
    四指上滑：F3
    四指水平滑动：切换 桌面/全屏程序
    三指聚合：F4
    三指散开：显示桌面
  6. 暂时想到这么多。。。

二、快捷键

贴一个我早前整理的一个表格：

cmd + w

关闭子窗口/标签页

不退出程序

cmd + q

关闭整个程序


cmd + tab

切换应用程序

相当于win的alt + tab

cmd + ~

切换程序下的子窗口

比如开了多个word文档，可以在这些文档间切换，很实用

cmd + 空格

切换输入法


在finder下，cmd + shift + g

呼出跳转目录对话框


option + 空格

呼出spotlight，常用于全局搜索文件

系统自动维护索引，搜索非常快，此快捷键可更改

在spotlight搜索结果中，选择某一项，cmd + 回车/鼠标单击

打开文件/目录所在的目录





这些是比较常用的，掌握越多，效率越高~

三、必备软件

  1. homebrew：是一个命令行包管理器，通过命令行管理（安装/卸载/更新）命令行类的软件，比如通过它安装git：
    brew install git

    非软件开发人员可能觉得命令行软件不是必须的，那么接下来的homebrew cask一定是你喜欢的菜。
    官网： http://brew.sh/index_zh-cn.html
  2. homebrew cask：它是基于homebrew的命令行工具，用于安装GUI程序，比如QQ、旺旺这些带界面的软件。回首传统的软件安装方式：打开浏览器，搜索，找一个dmg文件，下载，打开，拖到Applications，一个字——繁琐！
    而通过homebrew cask（以QQ为例）：打开终端，输入brew cask install qq，回车，Done！是不是很酷？！
    官网：http://caskroom.io/
  3. iTerm2：如果你是一名软件开发者，命令行一定少不了，快捷键，搜索神马的，非常强大，以弥补系统自带的terminal的不足。
    一键安装：brew cask install iterm2
    结合zsh这个shell，如虎添翼，参考卖桃君的教程 终极 Shell

  4. office：这个不必多言，依然一键安装：

    brew cask install microsoft-office

  5. shadowsocks：用于出墙：

    brew cask install shadowsocksx

    几乎所有的出墙方式，够需要与服务端配合，shadowsocks可参考我这篇文章 DIY零代码搭建 shadowsocks服务

  6. 目前国内的网银还是个问题，有些网银仍然只支持IE，所以你需要安装虚拟机来使用网银。那么VMware Fusion是一个好选择：

    brew cask install vmware-fusion

  7. 还有输入法，笔记软件，通讯软件等等，选择很多，我就不一一罗列了，暂时讲这么多

总结

熟悉linux的童鞋，上手OS X会非常快，从windows转过来的话，可能需要花点时间，半个月足矣。

另外，我也是受卖桃君影响，觉得他说的“同是程序员，本身就应该多多互相支持”(原话可能有出入)真的有道理，所以呢，不要希望什么软件都要免费，如果你觉得一个收费软件能够大大提高你的工作效率，或是能够真正满足你的需求，就放手大胆的买下它吧。

ps：我购买过的软件/服务有：

  1. 红杏：早期出墙chrome插件
  2. 搬瓦工VPS：主要用于shadowsocks出墙
  3. 衡天虚拟空间：用于本博客
  4. 衡天域名：用于本博客
  5. webstorm：用于日常工作
  6. viscosity：用于VPN客户端
